# Dzinehome Immersive Experience

A cinematic, scroll-driven concept for Dzinehome built as Fameboat's first digital-showroom demonstration.

## Experience highlights

- Full-screen image-led opening
- Smooth scrolling and parallax movement
- Interactive interior-style selector
- Immersive 212 NSL Residence story
- Scroll-pinned horizontal project showcase
- Dzinehome 5D process presentation
- Responsive mobile and desktop layouts
- Reduced-motion accessibility support

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Netlify

The repository includes `netlify.toml`.

- Build command: `npm run build`
- Publish directory: `dist`

## Content note

This is a concept experience using images and project references already published on dzinehome.in. It is intended to demonstrate a future immersive direction. Original source assets can replace the web-optimised images before a full production launch.
